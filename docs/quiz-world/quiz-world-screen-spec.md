# 通知型早押しクイズワールド MVP画面仕様

## 目的

このドキュメントは、実装前にMVPの画面単位の仕様を具体化するためのものである。
実装コードは作らず、画面の目的、表示要素、操作、権限、空状態、エラー状態、MVPでやらないことを整理する。

## 共通方針

- MVPは1ワールド制。
- Season 0は管理者発行の招待コード制。
- 18歳以上限定。
- 生年月日は保存しない。
- 四択クイズを優先する。
- 通知Phase 1では `/home` の画面内通知で検証する。
- start_at 到達前は問題を表示しない。
- 回答順位はサーバー側で決まる。
- タイムライン、コメント、DM、フォローは入れない。

## `/`

| 項目 | 内容 |
| --- | --- |
| 目的 | サービス概要を伝え、ログインまたは登録へ誘導する。 |
| 権限 | 未ログイン、ログイン済みの両方。 |

### 表示要素

- サービス名
- 「通知型早押しクイズワールド」の短い説明
- 現在の参加枠とシーズンの概要
- 18歳以上限定の説明
- ログインボタン
- 登録ボタン
- 招待コードが必要である説明

### 主な操作

- `/login` へ移動
- `/signup` へ移動

### 遷移先

- `/login`
- `/signup`
- ログイン済みなら `/home`

### 空状態

- world情報が取得できない場合は、概要だけ表示し、登録ボタンは一時的に無効化する。

### エラー状態

- world情報取得失敗
- サービス一時停止

### MVPでやらないこと

- 公開タイムライン
- ユーザー一覧の全面公開
- ランキング詳細の一般公開

## `/login`

| 項目 | 内容 |
| --- | --- |
| 目的 | 登録済みユーザーがログインする。 |
| 権限 | 未ログイン。 |

### 表示要素

- ログインフォーム
- パスワードリセット導線
- 登録導線
- 利用規約とプライバシーポリシーへのリンク

### 主な操作

- ログイン
- `/signup` へ移動
- パスワードリセット

### 遷移先

- ログイン成功時は `/home`
- 登録導線は `/signup`

### 空状態

- なし。

### エラー状態

- 認証失敗
- 停止中ユーザー
- 18歳以上確認や規約同意が未完了の既存ユーザー

### MVPでやらないこと

- SNS的なプロフィール検索
- 外部SNS連携の必須化

## `/signup`

| 項目 | 内容 |
| --- | --- |
| 目的 | 招待コードを持つ18歳以上のユーザーが登録する。 |
| 権限 | 未ログイン。 |

### 表示要素

- 表示名入力
- メールまたは認証情報入力
- 招待コード入力
- 18歳以上チェック
- 利用規約同意チェック
- プライバシーポリシー同意チェック
- 登録ボタン
- 満員時のwaitlist導線
- 生年月日は保存しない旨

### 主な操作

- 招待コード検証
- 登録
- waitlist登録へ移動

### 遷移先

- 登録成功時は `/home`
- 満員時は `/invite` のwaitlist表示
- ログイン済みなら `/home`

### 権限

- 招待コードを持つ未ログインユーザーのみ登録可能。
- 招待コードの検証はサーバー側で行う。

### 空状態

- 招待コード未入力
- 必須チェック未完了

### エラー状態

- 招待コードが無効
- 招待コード期限切れ
- 招待コード使用済み
- 18歳以上チェック未完了
- 利用規約同意未完了
- プライバシーポリシー同意未完了
- 参加枠満員
- 登録処理失敗

### 保存方針

- 生年月日はMVPでは保存しない。
- age_confirmed_at を保存する。
- terms_accepted_at を保存する。
- privacy_accepted_at を保存する。

### MVPでやらないこと

- 保護者同意フロー
- 16〜17歳対応
- 生年月日の保存
- 一般ユーザーによる招待コード発行

## `/home`

| 項目 | 内容 |
| --- | --- |
| 目的 | 届いたクイズ、回答状況、出題導線、ランク概要を表示する。 |
| 権限 | ログイン済み、world member。 |

### 表示要素

- 届いたクイズ一覧
- 開始前のクイズ
- 回答可能なクイズ
- 回答済みクイズ
- 終了済みクイズ
- 出題ボタン
- 自分の今日の残り出題回数
- 通知モード表示
- 出題ランク概要
- 回答ランク概要
- worldの参加枠とシーズン概要

### 主な操作

- クイズを開く
- 出題画面へ移動
- 結果画面へ移動
- プロフィールへ移動
- world画面へ移動

### 遷移先

- `/quiz/[launchId]`
- `/result/[launchId]`
- `/create`
- `/profile`
- `/world`

### 権限

- quiz_recipients に含まれるクイズだけ表示する。
- 停止中ユーザーは出題・回答不可。

### 空状態

- 届いたクイズがない
- 今日の出題回数が残っていない
- まだ回答履歴がない

### エラー状態

- クイズ一覧取得失敗
- world member ではない
- アカウント停止中

### MVPでやらないこと

- タイムライン
- コメント
- DM
- フォロー投稿
- 全ユーザーの公開活動フィード

## `/create`

| 項目 | 内容 |
| --- | --- |
| 目的 | 出題者が四択クイズを作成し、配信する。 |
| 権限 | ログイン済み、world member、停止されていないユーザー。 |

### 表示要素

- 問題文入力
- 選択肢4つ
- 正解選択肢
- 難易度
- カテゴリ
- 出題ボタン
- 今日の残り出題回数
- 今回の配信人数
- 出題者ランク
- 不適切内容への注意文
- 出題対象抽選はサーバー側で行う説明

### 主な操作

- 下書き入力
- 四択クイズ作成
- 出題

### 遷移先

- 出題成功時は `/quiz/[launchId]` または `/home`
- エラー時は同画面に留まる

### 権限

- 出題者ランクに応じた1日の出題回数と配信人数をサーバー側で判定する。
- 出題対象者はクライアントで指定させない。

### 空状態

- 問題文未入力
- 選択肢不足
- 正解未選択

### エラー状態

- 今日の出題回数上限に達している
- 選択肢が重複している
- 問題文が短すぎる
- 不適切内容の疑い
- 配信可能対象者がいない
- アカウント停止中

### MVPでやらないこと

- 自由入力のみの問題
- AI自動判定
- 画像問題
- 出題対象者の手動選択
- 課金による配信人数増加

## `/quiz/[launchId]`

| 項目 | 内容 |
| --- | --- |
| 目的 | 通知対象者がカウントダウン後に回答する。 |
| 権限 | quiz_recipients に含まれるログイン済みユーザー。 |

### 表示要素

- カウントダウン
- start_at 到達前の待機表示
- start_at 到達後の問題文
- 四択選択肢
- 回答送信ボタン
- 回答済み状態
- 締切済み状態
- サーバー側で順位が決まる説明

### 主な操作

- 回答選択
- 回答送信
- 結果へ移動

### 遷移先

- 回答後は `/result/[launchId]`
- 締切後も `/result/[launchId]`

### 権限

- 自分が配信対象でない場合は閲覧不可。
- 出題者本人は回答不可。

### 空状態

- start_at 前は問題を表示せず、カウントダウンのみ表示する。

### エラー状態

- 配信対象ではない
- launchが存在しない
- launchがキャンセル済み
- end_at後で回答不可
- すでに回答済み
- 通信失敗

### MVPでやらないこと

- start_at前の問題先出し
- 複数回答
- 回答後の選び直し
- クライアント時刻による順位判定

## `/result/[launchId]`

| 項目 | 内容 |
| --- | --- |
| 目的 | 回答結果、順位、全体結果、評価導線を表示する。 |
| 権限 | 出題者、配信対象者、admin。 |

### 表示要素

- 自分の正誤
- answer_rank
- correct_rank
- 全回答者一覧
- 未回答者
- 正解者順位
- クイズ評価
- 理由タグ
- 通報導線
- 出題者名
- 問題の難易度
- 問題のカテゴリ
- 正解選択肢
- 正解率

### 主な操作

- 評価を送信
- 理由タグを選択
- 通報
- `/home` へ戻る

### 遷移先

- `/home`
- `/profile`
- 通報送信後は同画面

### 権限

- 回答前は他人のanswersを見せない。
- 終了後または回答後に、対象者へ結果として公開する。

### 空状態

- まだ回答者がいない
- 未回答者のみ
- 評価未送信

### エラー状態

- launchが存在しない
- 結果閲覧権限がない
- 評価済み
- 通報送信失敗

### MVPでやらないこと

- コメント欄
- DM送信
- 自由チャット
- SNS共有機能の作り込み

## `/profile`

| 項目 | 内容 |
| --- | --- |
| 目的 | 自分のランク、スコア、通知設定、同意状態を管理する。 |
| 権限 | ログイン済み本人。 |

### 表示要素

- 表示名
- アバター
- 出題ランク
- 回答ランク
- 出題スコア
- 回答スコア
- 通知設定

- 1日の通知上限
- ブロック一覧
- 利用規約同意状態

### 主な操作

- 表示名変更
- 通知モード変更
- quiet hours変更

- 1日の通知上限変更
- ブロック解除

- フォロー一覧
- DM導線
- 生年月日の登録

## `/admin`

| 項目 | 内容 |
| --- | --- |
| 目的 | 10人テストを安全に運用する。 |
| 権限 | admin role のみ。 |

### 表示要素

- 通報一覧
- 通報詳細
- クイズ配信停止操作
- ユーザー停止操作
- waitlist一覧
- 招待コード発行
- 参加枠調整
- world状態確認
- ランクイベント確認
- 通知ログ確認
- 操作ログ
- 誤操作防止の確認表示

### 主な操作

- 通報ステータス更新
- question.status を suspended にする
- user.status を suspended にする
- waitlist承認
- 招待コード発行
- member_limit変更
- 操作ログ確認

### 遷移先

- `/admin`
- `/home`
- 通報詳細ビュー

### 権限

- admin roleのみアクセス可能。
- admin APIもadmin roleのみ許可する。

### 空状態

- 通報がない
- waitlistが空
- 通知ログがない
- 操作ログがない

### エラー状態

- admin権限がない
- 対象クイズが存在しない
- 対象ユーザーが存在しない
- 参加枠が現在参加人数より小さい
- 操作ログ記録失敗

### MVPでやらないこと

- 完全削除
- cloud dataの直接手動削除前提の運用
- Stripe管理
- Vercel env管理
- Supabase project管理
